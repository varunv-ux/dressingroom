# Chrome Web Store Listing

## Name
Dressing Room

## Short Description
Create AI try-ons on shopping pages and save your favorite looks.

## Detailed Description
Dressing Room adds a lightweight AI try-on layer to shopping pages. Add a reference photo, paste your own OpenAI API key, and generate try-on images from product photos while keeping the original shopping page intact.

Generated looks are collected in a clean Dressing room where you can filter by brand, rearrange cards, move favorites to a side rail, hide looks, and reopen the original product page.

Features:
- Try on visible product photos from supported shopping pages
- Save generated looks in your browser
- Filter looks by brand
- Drag and rearrange look cards
- Move favorite looks into a side rail
- Bring your own OpenAI API key
- Generated images are stored through the hosted Dressing Room API and Vercel Blob

Privacy and keys:
Your OpenAI API key is stored locally in Chrome extension storage and is sent only to the Dressing Room generation API when you request a try-on. The extension does not store your key on the server.
